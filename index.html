Projects

  • Diabetic Retinopathy Detection in Low Resolution Retinal Images for Low Resource Environments

    Designed a preprocessing pipeline for retina images from two different sources to be used to train a deep learning model for diabetic retinopathy detection. Set up a training environment on a distributed compute cluster and used SLURM to schedule training jobs. Trained and evaluated the performance of a transfer learning model based on InceptionV3 for diabetic retinopathy detection.

  • Mental Stress Level Prediction from Mobile Phone Biomarkers

    Created a mental stress data-to-image conversion pipeline that enabled the use of explainability techniques used for images in interpreting stress data from study subjects. Built a deep learning pipeline to predict the stress level of stress study subjects on a scale of 0 to 10 using data from phones and wearables. Used different explainability techniques such as class activation maps and local interpretable model-agnostic explanations (LIME) to highlight the areas of each image that were most impactful in the model prediction

  • DigiSketch: A Collaborative Drawing System using FPGAs

    Implemented a drawing system using two FPGAs communicating through I2C. The system allowed real-time collaboration between two drawers, allowing one to save sketches to permanent memory, and/or use saved sketches as templates for new sketches.
